# HON商店 Shopifyストア更新スクリプト

`tsukutte-uru.myshopify.com` のコンテンツ・デザイン更新ツール。

---

## セットアップ

### 1. APIトークンを取得する

1. Shopify管理画面 → **設定** → **アプリと販売チャネル**
2. **アプリを開発する** → **アプリを作成**
3. アプリ名: `HON-Scripts`（任意）
4. **管理API権限**を設定:
   - テーマ: 読み取り / 書き込み
   - 商品: 読み取り / 書き込み
   - コレクション: 読み取り / 書き込み
   - ページ: 読み取り / 書き込み
   - ナビゲーション: 読み取り / 書き込み
5. **アプリをインストール** → アクセストークンをコピー

### 2. .env ファイルを作成

```bash
cp .env.example .env
# .env を開いてトークンを貼り付け
```

### 3. Node.js バージョン確認（v18以上）

```bash
node --version  # v22.x.x であればOK
```

---

## スクリプト実行

### 全スクリプトを順番に実行（推奨）

```bash
npm run all
```

### 個別実行

```bash
# トップページ更新（コンテンツ8セクション + 和レトロモダンCSS）
npm run homepage

# 新商品3点を追加
npm run products

# ナビゲーション更新
npm run nav
```

## 注意事項

- スクリプトはトップページ（`templates/index.json`）を**上書き**します
- 実行前にShopify管理画面でテーマを複製してバックアップを推奨
- テーマが `custom-liquid` セクションタイプに対応している必要があります
  （Horizonテーマは対応済み）
