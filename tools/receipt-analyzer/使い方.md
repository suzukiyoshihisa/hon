# 領収書解析ツール 使い方

レシート写真をAIで読み取り、経費CSVを自動生成するツールです。

---

## 毎月の手順

### 1. レシート画像をフォルダにまとめる

どこのフォルダでも構いません。画像をまとめて入れるだけです。

```
例: /Users/suzukiyoshihisa/Desktop/領収書/2026/
    ├── IMG_001.jpg
    ├── IMG_002.jpg
    └── ...
```

対応形式: **JPG / PNG / WEBP / HEIC / GIF**

### 2. ターミナルで run.sh にフォルダを渡す

```bash
"/Users/suzukiyoshihisa/Desktop/ 株式会社 HON/tools/receipt-analyzer/run.sh" \
  "/Users/suzukiyoshihisa/Desktop/領収書/2026"
```

**ターミナルで `run.sh` まで入力してから、Finderのフォルダをドラッグ＆ドロップするとパスが自動入力されます。**

### 3. output フォルダに CSV が生成される

```
receipt-analyzer/output/
└── 2026_receipts_20260313.csv
```

### 4. Google Sheets にインポートする

1. Google Sheets を開く
2. **ファイル → インポート → アップロード**
3. 生成された CSV を選択
4. 区切り文字: **カンマ** を選択してインポート

---

## CSV の列構成

| 列 | 内容 | 備考 |
|----|------|------|
| A | ファイル名 | 原本突合用 |
| B | 日付 | YYYY-MM-DD |
| C | 支払先 | 店舗・会社名 |
| D | 金額（税込） | 円 |
| E | 消費税額 | 円 |
| F | 金額（税抜） | D - E |
| G | 支払方法 | 現金/カード/電子マネーなど |
| H | 勘定科目 | 自動分類（下記参照） |
| I | 摘要 | 品目メモ |
| J | 信頼度 | high / medium / low |
| K | エラー | 読み取り失敗時のメッセージ |

---

## 自動分類される勘定科目

交通費 / 旅費交通費 / 接待交際費 / 会議費 / 消耗品費 / 通信費 /
広告宣伝費 / 新聞図書費 / 研修費 / 地代家賃 / 水道光熱費 /
福利厚生費 / 外注費 / 支払手数料 / 雑費 / 不明

---

## オプション

| オプション | 説明 |
|-----------|------|
| `--model sonnet` | 高精度モード（手書き・不鮮明な画像に有効） |
| `--dry-run` | ファイル一覧の確認のみ（APIを呼ばない） |

### 例

```bash
# 件数だけ確認したい（APIを呼ばない）
./run.sh /path/to/receipts/2026 --dry-run

# 通常実行
./run.sh /path/to/receipts/2026

# 手書きレシートが多い場合（精度重視・コスト高め）
./run.sh /path/to/receipts/2026 --model sonnet
```

---

## 信頼度が low の行は目視確認を

- **high**: 日付・金額・支払先がすべて読み取れた
- **medium**: 一部が読み取りづらいが主要情報は取得できた
- **low**: 画像が不鮮明、または必須情報が欠落している → 手動修正

## 年が変わったとき

```bash
mkdir receipts/2027
# 画像を入れてから
node src/index.js --year 2027
```

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| `ANTHROPIC_API_KEY を設定してください` | `.env` ファイルが存在するか確認 |
| `フォルダが見つかりません` | `receipts/2026/` フォルダが存在するか確認 |
| 文字化けする | Google Sheets インポート時の文字コードを **UTF-8** に設定 |
| 金額が 0 になる | 画像が不鮮明な可能性 → `--model sonnet` で再実行 |
