# wansie EC — サイト構想書

**作成日：** 2026-06-16
**作成者：** 株式会社HON
**ステータス：** 構想フェーズ

---

## 1. 概要

| 項目 | 内容 |
|------|------|
| サイト名 | wansie EC |
| プラットフォーム | Shopify |
| ターゲット | BNIメンバー・経営者（ECに挑戦中 or 検討中） |
| デザイン | ポップ・親しみやすい（明るいカラー × 丸みのあるUI） |
| 主な目的 | EC相談の入口 → TimeRexで日程調整 → 継続サポート契約 |
| 主担当者 | 鈴木（株式会社mid） |

---

## 2. コンセプト

> **「ECで詰まったら、まずwansieに来てください。」**

ECサイトを自力で作ろうとして挫折する人・売れずに困っている経営者に向けて、
「相談できる場所がある」という安心感を与えるサービスサイト。

TimeRexで気軽に相談予約できる入口を前面に出しながら、
サービス内容・料金・鈴木のプロフィール・商品購入まで一気通貫で案内する。

## 4. 各セクション詳細

### 4-1. ファーストビュー

**キャッチコピー（案）:**
```
ECのこと、全部相談してください。
```

**サブコピー（案）:**
```
商品づくり・ショップ開設・売上アップ、プロが並走します。
まずは気軽にお話しましょう。
```

**CTA ボタン：** `無料相談の日程を選ぶ` → TimeRex URL

**ビジュアルイメージ：** ポップなイラスト or 明るい笑顔の写真。
背景カラー：明るいオレンジ or 黄色系。

---

### 4-2. お悩みリスト（こんなことで困っていませんか？）

6つの悩みカード（絵文字アイコン付き）

| # | 悩み |
|---|------|
| 1 | ECショップを作りたいけど、何から始めればいいか分からない |
| 2 | Shopifyを使ってみたけど設定が難しくて止まっている |
| 3 | ショップを作ったのに全然売れない |
| 4 | 売れる商品のアイデアが浮かばない |
| 5 | 外注に頼んだら高くて、その後のことを教えてもらえなかった |
| 6 | BNIで紹介したいが、自社の商品・サービスがECに向いているか不安 |

---

### 4-3. EC悩み相談（メインサービス）

**見出し：** 「まず話してみましょう。相談だけでもOKです。」

**内容：**
- 30分の無料相談
- オンライン（Zoom / Google Meet）
- TimeRexで24時間いつでも予約可能
- 相談後に売り込みはしない（安心感の一言）

**CTA：** `TimeRexで日程を選ぶ` → [TimeRex URL を後で設定]

**TimeRex連携方法（実装時）：**
- TimeRex のカレンダーリンクをボタンに設定
- または TimeRex の埋め込みウィジェットを iframe で挿入

---

### 4-4. プロと並走するとこんなに簡単（ステップ紹介）

**見出し：** 「一人でやると3ヶ月かかることが、プロと一緒なら2週間で。」

**ステップ（5段階）：**

| STEP | 内容 |
|------|------|
| 01 | 無料相談で現状ヒアリング（30分） |
| 02 | やりたいことを整理→プラン提案 |
| 03 | ショップ設計・商品ページ制作 |
| 04 | 決済・配送・導線の設定 |
| 05 | 公開！その後も並走サポート |

**ポイント訴求（3つ）：**
- 自分でできるように教えながら進める
- Shopifyの知識がゼロでも大丈夫
- 途中でやめてもOK（押し売りなし）

---

### 4-5. 商品リスト

**見出し：** 「まずは使ってみる。試せる商品を用意しました。」

Shopify の商品コレクションを表示。
デモ用 or 実販売商品を並べる。

**想定商品：**

| 商品名 | 種別 | 価格 | 目的 |
|--------|------|------|------|
| ECスタートアップガイド | デジタルDL（PDF） | ¥3,300 | 自走層向け入門コンテンツ |
| 無料相談予約 | サービス（¥0） | 無料 | 相談導線のサンプル |
| EC並走サポート（ミニ） | サービス | ¥30,000〜 | 有料相談・スポットサポート |

---

### 4-6. 料金プラン（ステージ別）

**見出し：** 「あなたのステージに合わせて選べます。」

**サブコピー：** まずは話してみて、ピッタリのプランを一緒に考えます。

| プラン名 | ステージ | 内容 | 価格 |
|---------|---------|------|------|
| 🌱 スタートプラン | EC未経験・これから始める | 無料相談 → 診断 → セットアップ支援 | ¥50,000〜 |
| 🚀 成長プラン | ショップはある・売れていない | 現状分析 → 改善提案 → 月次並走 | ¥30,000/月〜 |
| 🏆 拡大プラン | 売上あり・次のステージへ | 商品開発 + 販路拡大 + OEM支援 | 要相談 |

**共通特典：** 初回相談無料 / 強引な営業なし

---

### 4-7. 商品開発もお任せください

**見出し：** 「売るものがない？一緒に作ればいい。」

**サブコピー：**
```
アイデアはあるけど形にできない。
そんな方の商品開発を0から並走します。
```

**できること一覧：**
- 商品コンセプト企画
- OEM・小ロット製造先の紹介・交渉
- パッケージ・ネーミング提案
- EC販売ページの制作
- 販売開始後のPDCA支援

**実績ジャンル（例）：**
コスメ / 食品 / 雑貨 / ノベルティ / ウェルネス系 etc.

**CTA：** `商品開発について相談する` → TimeRex

---

### 4-8. 鈴木 / 株式会社midについて

**見出し：** 「担当するのはこの人です。」

**プロフィール（文案）：**
```
鈴木 ○○（すずき ○○）
株式会社mid 代表

Shopifyを中心としたECコンサルタント。
中小企業・個人事業主のEC立ち上げ〜運用改善を多数支援。
「難しいことをやさしく、やさしいことを楽しく」をモットーに、
EC初心者でも自走できる環境づくりにこだわっています。

BNIメンバーとして地域経営者のネットワークにも深く関わる。
```

**掲載要素：**
- プロフィール写真
- 経歴・実績（件数・業種など）
- BNIメンバーである旨（信頼感）
- SNS / 会社サイトへのリンク

## 6. デザイン方針

| 要素 | 方針 |
|------|------|
| カラー | メイン：コーラルオレンジ（#FF6B6B）or 明るい黄色（#FFD93D）。白ベース。 |
| フォント | 見出し：丸ゴシック系。本文：読みやすいサンセリフ。 |
| トーン | 絵文字・アイコン多用。カード型レイアウト。余白広め。 |
| 写真 | 笑顔・明るい・対話感。イラストとの組み合わせもOK。 |
| Shopifyテーマ候補 | Dawn（シンプルでカスタマイズしやすい）or Craft |
