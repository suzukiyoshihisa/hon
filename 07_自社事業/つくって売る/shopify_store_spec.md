## 6. 技術構成

### 開発環境

| ツール | バージョン |
|--------|---------|
| Node.js | v22.22.3（nvm管理） |
| Shopify CLI | 4.1.0 |
| Shopify AI Toolkit (MCP) | @shopify/dev-mcp 最新版 |

### API

| 種別 | 用途 |
|------|------|
| Shopify Admin REST API 2024-01 | 商品・ページ・テーマ操作 |
| Shopify Admin GraphQL API 2024-01 | メニュー操作 |

### セットアップスクリプト（`tools/tsukutte-uru/scripts/`）

| ファイル | 内容 | 状態 |
|---------|------|------|
| `update-homepage.js` | トップページ8セクション構成 + 和レトロモダンCSS | 作成済み |
| `add-products.js` | デモ商品3点追加 | 作成済み |
| `update-nav.js` | ナビゲーション更新 | 作成済み |

**実行方法:**
```bash
cd "tools/tsukutte-uru"
cp .env.example .env  # → トークンを設定
npm run all           # 全スクリプト一括実行
```

## 7. 残タスク

### 優先度：高（スクリプト実行前に必要）
| タスク | 内容 |
|--------|------|
| Admin APIトークン取得 | Shopify管理画面 → 設定 → アプリ → カスタムアプリ作成 |
| .env 設定 | tsukutte-uru/.env.example を .env にコピーして記入 |
| スクリプト実行 | `npm run all` で商品追加・ナビ更新・トップページ更新 |

### 優先度：高（スクリプト実行後）
| タスク | 内容 |
|--------|------|
| テーマバックアップ | スクリプト実行前にShopify管理画面でテーマを複製 |
| 商品画像の追加 | 5商品すべてにビジュアルを追加 |
| Calendly埋め込み設定 | 相談予約ページに実装 |
| 決済設定 | Shopify Payments または他の決済手段 |

### 優先度：中
| タスク | 内容 |
|--------|------|
| アナウンスバーの日本語化 | テーマエディターから手動設定 |
| フッター日本語化 | 「Join our email list」→「メールマガジン登録」など |
| ドメイン設定 | 本番運用時に独自ドメインを設定 |
| タイムゾーン・地域設定 | 設定 → 全般 で日本に変更 |

## 8. 構築フロー（再現手順）

1. `nvm use 22`
2. `npx @shopify/cli@latest auth login`
3. `.env` に `SHOPIFY_ACCESS_TOKEN` と `SHOPIFY_STORE` を記載
4. `node --env-file=.env scripts/setup-store.js`
5. `node --env-file=.env scripts/setup-products.js`
6. `node --env-file=.env scripts/setup-homepage.js`
7. `node --env-file=.env scripts/finalize-store.js`
8. `node --env-file=.env scripts/cleanup-pages.js`
