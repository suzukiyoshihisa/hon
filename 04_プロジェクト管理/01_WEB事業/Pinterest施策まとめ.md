# Pinterest 施策まとめ

- **作成日**: 2026-07-02
- **対象サイト**: wansie オンラインショップ / wansie ユニフォーム

---

## 1. なぜ今Pinterestか

### 現状の課題（アナリティクスより）

| サイト | 現状の課題 | Pinterestで解決できること |
|--------|-----------|------------------------|
| wansie (EC) | ノンブランド検索がほぼゼロ。「wansieを知らない人」に届いていない | ブランド未認知層への視覚的発見 |
| wansie (EC) | トップページ止まり52%。回遊されていない | 商品・コレクションページへの直接流入 |
| wansie uniform | Organic Socialが10セッションのみ。SNS流入が機能していない | BtoB顧客（美容室オーナー等）へのリーチ |

### Pinterestが有効な理由

1. **購買意欲の高いユーザーが集まる** — Pinterestは「欲しいものを探す場所」。InstagramやTikTokと違いエンタメ消費ではなく「購入・発注の検討」文脈で使われる
2. **ピンの寿命が長い** — 投稿したピンは数ヶ月〜数年単位で流入をもたらす。SNSの投稿が数時間で埋もれるのと対照的
3. **フランス語圏との相性** — wansieの `/fr` ページはCTR25%と高く、フランス語圏ユーザーの関心が確認されている。Pinterestはフランスでの利用率が特に高い
4. **Shopify連携が強力** — 商品カタログを自動でピン化し、価格・在庫情報をリアルタイム反映する「ショッピングピン」が使える

## 2. サイト別 適性・優先度

### wansie オンラインショップ（優先度: 高）

- **適性**: ◎ ファッション・アパレルはPinterestの最強コンテンツジャンル
- **勝ちパターン**: コレクション（FW/SS）単位のボード × 商品ページへの直リンク
- **特記**: `/fr` のCTR実績からフランス語圏向けボードを別途設けると効果的

### wansie ユニフォーム（優先度: 中）

- **適性**: ○ 美容室・ホテル・カフェなどの「おしゃれな制服」はPinterestで検索されやすいニッチ
- **勝ちパターン**: 業種別の施工事例ボード（美容室・ホテル・etc.）
- **ターゲット**: 美容室オーナー・店長・内装担当者。Pinterestでサロンインテリアと一緒にユニフォームも探す層

### mid_group（優先度: 低）

- コーポレートサイトのためPinterestとの親和性は低い。現時点では優先しない

## 3. アカウント設定（準備項目）

### Step 1: ビジネスアカウント作成・切り替え
- `pinterest.com/business` からビジネスアカウントへ切り替え（または新規作成）
- ビジネスアカウントで使えること: アナリティクス / 広告 / ショッピング連携

### Step 2: プロフィール設定
```
アカウント名   : wansie（ブランド名そのまま）
プロフィール  : 東京発のファッションブランド / オリジナルユニフォーム制作
Webサイト    : https://store.wansie.tokyo/
               ※ドメイン認証を必ず実施（SEO効果に直結）
```

### Step 3: Shopify × Pinterest 連携（wansie EC）
1. Shopify管理画面 → 「販売チャネル」→「Pinterest」を追加
2. 商品カタログが自動でPinterestに同期され、**ショッピングピン**（価格・在庫情報付き）になる
3. ピンをクリックすると商品ページに直接遷移するため、コレクションへの回遊率改善に直結する

## 5. コンテンツ戦略

### 投稿頻度・タイミング
- **推奨頻度**: 週5〜10ピン（毎日少量ずつが理想。まとめて投稿はNG）
- **推奨時間帯**: 夜20〜22時（日本のユーザー）/ 午前中（フランス語圏向け）
- **ツール**: Tailwind（Pinterest公式パートナー）でスケジュール投稿が可能

### コンテンツ種別と優先度

| 種別 | 説明 | 優先度 |
|------|------|-------|
| 商品ピン（ショッピングピン） | Shopify連携で自動生成。価格・リンク付き | 最高 |
| コレクションルック | モデル着用・スタイリング写真 | 高 |
| コーデ提案ピン | 複数商品の組み合わせ提案（縦長1:2比率推奨） | 高 |
| 制作プロセス動画 | ユニフォームの生地選び・サンプル工程 | 中 |
| ムードボード | ブランドの世界観・カラーパレット | 中 |
| テキストオーバーレイピン | 「美容室制服のトレンド2026」など情報系 | 低〜中 |

### 画像仕様（重要）

```
推奨サイズ : 1000 × 1500px（縦長2:3比率）
形式       : JPEG または PNG
テキスト   : ピン上部か下部に商品名・ブランド名を入れると保存時に認識されやすい
URL        : 必ずトップページではなく商品・コレクションページに直リンク
```

## 6. フランス語圏向け施策（wansie 専用）

アナリティクスで `/fr` のCTR25.35%という高いシグナルがあるため、専用対応を推奨。

1. **フランス語ボードを別途作成** — `Mode Parisienne` / `Style Japonais` などのワードは仏語圏で検索される
2. **ピンの説明文をフランス語で記載** — Google翻訳でも十分。フランス語ハッシュタグを付加
3. **`/fr` 版商品ページへの直リンク** — フランス語版URLに誘導してCVRを高める
4. **フランス語ハッシュタグ例**:
   ```
   #modejaponaise #stylejaponais #salopette #pantalonslarges #tenuedujour #modeparis
   ```

## 9. 注意点・リスク

- **すぐには効果が出ない** — Pinterestは育てるプラットフォーム。流入が実感できるのは2〜3ヶ月後が目安
- **ピンの質 > 量** — 雑多な投稿よりも縦長・高解像度・商品ページ直リンクの質の高いピンに集中する
- **ショッピングピンの審査** — Shopify連携後、Pinterestの商品カタログ審査（数日）が必要
- **ビジネスアカウント必須** — 個人アカウントのままではアナリティクスもショッピング機能も使えない
