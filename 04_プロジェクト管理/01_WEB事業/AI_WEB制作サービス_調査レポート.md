# AI × WEB制作サービス 新規事業調査レポート

調査日: 2026-03-05
担当: 鈴木

---

## 概要

Claude Code と BNI コネクションを活用した WEB制作・ECサイト制作サービスの事業化調査。
提案から制作・保守まで一気通貫で担うサービスモデルを検討する。

---

## 1. 市場トレンド（2025〜2026年）

### WEB制作 × AI の現状

| 指標 | データ |
|------|--------|
| WEBデザイナーの単価上昇実感 | 62.7%が「上がった」 |
| 案件受託量の増加 | 74%超が「増加」 |
| AI活用で修正回数が減少 | 63.2%が実感 |
| AI活用で品質が向上 | 47.2%が実感 |

### なぜ今チャンスか

- Claude Code / Cursor の普及により開発速度が 3〜5倍 に向上
- 納期短縮 → 原価率が下がる → 同価格で利益率が跳ね上がる
- AIを使えない制作会社との「差」が急速に拡大中
- 2025年案件受託量は「大幅に増えた」21.8%、「やや増えた」52.7%

---

## 2. BNI コネクション活用

### BNI とは

BNI（Business Network International）は世界最大級のビジネス・リファーラル組織。
- 世界70ヶ国以上、11,000超チャプター、会員数31万人
- 日本: 2006年開始、国内会員12,000人超
- **1専門分野1名制** → WEB制作枠で入れば独占ポジション

### BNI の実績データ

- 60名規模チャプター → 7年間で10億円超のリファーラル
- Web制作 × IT補助金パッケージで年2,500万円の事例
- ロゴ制作案件が1.5万円 → 30万円に増額した事例（紹介営業の威力）

### BNI 活用の勝ち筋

| BNIでやること | Claude Codeでやること |
|--------------|---------------------|
| 異業種の中小企業経営者に接触 | 提案書・モックアップを爆速で生成 |
| 「ホームページ困ってる」案件を紹介してもらう | 制作工数を大幅削減して利益確保 |
| 保険・税理士・コンサルと相互紹介 | 保守案件を継続的に積み上げ |

## 3. サービス設計

### フロー

```
【提案】BNIリファーラルで受注（広告費ゼロ）
    ↓
【制作】Claude Code + Cursor で高速開発
    ↓
【保守】月額ストック収益で積み上げ
```

### 料金メニュー（市場相場ベース）

| メニュー | 相場 | AI活用後の工数圧縮 |
|---------|------|------------------|
| LP制作（1ページ） | 15〜50万円 | 1/3 |
| コーポレートサイト（5〜10P） | 50〜150万円 | 1/2 |
| ECサイト（Shopify等） | 80〜300万円 | 1/2〜1/3 |
| 月額保守・更新 | 月3〜5万円/件 | ← ストック収益の柱 |
| AIチャットボット付きオプション | +30〜80万円 | 差別化オプション |

### 原価

- Claude Code Pro: $20/月（約3,000円）
- API費用: 案件あたり数千〜1万円程度
- 利益率目安: **40〜60%**

## 5. 差別化ポイント

1. **スピード**: 通常2〜3ヶ月 → 2〜3週間で納品
2. **AI付加価値**: チャットボット・SEO自動生成をデフォルト提案
3. **BNI紹介網**: 広告費ゼロで温かいリードを獲得
4. **HONブランド**: コンサル実績（カオリウム等）を信頼担保に活用
