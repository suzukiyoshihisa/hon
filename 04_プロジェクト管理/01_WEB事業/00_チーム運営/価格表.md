## 基本方針

- Claude Code × Cursor による高速開発で、市場相場より低コスト・短納期を実現
- AI活用により制作工数を **1/2〜1/3** に圧縮 → 利益率 **40〜60%** を確保
- 受注価格は案件規模・クライアント属性・BNIリファーラル有無で調整する

## 5. 受注判断チェックリスト

- [ ] 予算感がHON価格帯に合っているか
- [ ] 制作期間に余裕があるか（緊急案件は割増対応）
- [ ] 素材（画像・テキスト・ロゴ）をクライアントが用意できるか
- [ ] ドメイン・サーバーの手配方針を確認済みか
- [ ] 月額保守への誘導チャンスがあるか

## 6. 値引き・割引ルール

| 条件 | 割引率 |
|------|--------|
| BNIリファーラル案件 | 5〜10%（関係構築優先） |
| 保守契約セット受注 | 5% |
| 複数サービス同時受注 | 最大10% |
| 既存クライアントの追加依頼 | 5% |

> **値引きの下限**: 利益率30%を下回る受注はしない
