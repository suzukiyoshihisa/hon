# DSEP 活動マニュアル
## 講義サポート業務

作成日: 2026-03-25
対象: DSEP運営スタッフ
作成: 株式会社HON 鈴木

---

## 1. DSEPについて

**DSEP（エグゼクティブ向け災害対策履修プログラム）** は、
企業経営者・エグゼクティブ層を対象とした災害対策の研修・講義プログラムです。

- 主催: 沼田教授
- 内容: 研修・講義

---

## 2. 講義サポート業務 全体フロー

```
【講義前】準備・確認
      ↓
【講義当日】会場設営・受付・進行補助
      ↓
【講義後】片付け・フォローアップ
```

---

## 3. 講義前の準備

### 1週間前
- [ ] 参加者リスト確認・人数把握
- [ ] 会場・設備の予約確認
- [ ] 配布資料の印刷・準備
- [ ] 講師（沼田教授）との最終確認

### 前日
- [ ] 機材動作確認（プロジェクター・PC・マイク等）
- [ ] 配布資料の部数確認
- [ ] 受付リスト・名札の準備
- [ ] 緊急連絡先リストの確認

### 当日（開始2時間前）
- [ ] 会場設営（座席配置・スクリーン・机）
- [ ] 機材セッティング・動作確認
- [ ] 受付テーブル準備
- [ ] 飲み物・備品の配置

| 役割 | 担当 | 業務内容 |
|------|------|---------|
| 受付 |  | 参加者チェック・名札配布・資料配布 |
| 進行補助 |  | 講師サポート・時間管理・スライド操作 |

| 記録 |  | 写真撮影・議事録 |

---

## 5. 講義中のサポート業務

### 受付
- 開始30分前から受付開始
- 参加者リストと照合し、チェックマークを記入
- 名札・資料を手渡す
- 遅刻者は静かに案内する

### 進行補助
- 講師の指示に従いスライドを操作
- 残り時間を講師に伝える（残り15分・5分のタイミング）
- 質疑応答時は挙手した参加者にマイクを渡す

### 機材担当
- 音声・映像トラブル時は即座に対応
- 録画・録音が必要な場合は事前に確認して対応
- トラブル発生時は進行を止めず、スタッフ間で対処する

---

## 6. 講義後の対応

### 終了直後
- [ ] 参加者のお見送り
- [ ] アンケートの回収（実施する場合）
- [ ] 名刺交換のサポート

### 片付け
- [ ] 配布資料の余部回収・廃棄
- [ ] 機材の撤収・返却
- [ ] 会場の原状回復
- [ ] 忘れ物確認

### 事後対応
- [ ] 参加者リスト（出欠）を沼田教授に報告
- [ ] アンケート結果の集計・共有
- [ ] 次回開催に向けた改善メモの作成

---

## 7. 緊急時対応

| 状況 | 対応 |
|------|------|
| 機材トラブル | 機材担当が即時対応。復旧まで講師に口頭進行を依頼 |
| 参加者の体調不良 | 最寄りの休憩スペースへ案内。必要に応じて救急対応 |
| 遅延・中断 | 沼田教授に状況を伝え、進行変更の指示を仰ぐ |
| 会場トラブル | 施設管理者へ連絡。参加者に状況をアナウンス |
