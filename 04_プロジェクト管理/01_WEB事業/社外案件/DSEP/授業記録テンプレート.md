# 授業記録テンプレート（2026年度）

## パターンA｜通常講義

> タイトル形式：「第○回前半/後半講義　講師：○○ 先生」

### タイトル
```
第〇回前半講義　講師：○○ ○○ 先生
```
※ 1回の授業が前半・後半に分かれる場合はそれぞれ別記事として作成

### 日付・場所
```
2026年〇月〇日（曜）〇〇:〇〇
東京大学本郷キャンパス
```

### 登壇者
```
○○ ○○（○○大学 ○○研究所 ○○学府 教授）
```
※ 所属は正式名称で記載（例：東京大学国際高等研究所カブリ数物連携宇宙研究機構、学際情報学府教授）

### 本文

```
2026年〇月〇日（曜）〇〇時より、東京大学本郷キャンパスにて、「災害対策エグゼクティブプログラム（DSEP）」第〇回前半/後半講義として、[講師の所属・肩書]の○○ ○○先生にお話しいただきました。

○○先生はご自身の研究分野と絡めて、「○○○○」などについて話していただきました。
```

### 写真
- 写真1〜：講義風景（複数枚・キャプションなし）
