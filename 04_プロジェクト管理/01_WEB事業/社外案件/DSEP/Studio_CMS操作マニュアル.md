# DSEP 授業記録 Studio CMS 操作マニュアル

対象: 2026年度 授業記録の追加・公開作業
作成: 株式会社HON 鈴木
更新日: 2026-04-26

---

## 担当者と役割

| 担当 | 役割 |
|------|------|
| 授業サポートスタッフ | 授業当日のサポート・写真撮影補助 |
| カメラマン | 授業写真の撮影・共有 |
| **鈴木（HON）** | CMS記事作成・Slack（運用）への共有 |
| **中野さん** | テキスト確認・修正 |
| — | Slack（全体）への最終共有 |

---

## 全体の流れ

```
【授業当日】
授業サポートスタッフが授業をサポート
        ↓
カメラマンが写真を撮影・鈴木に共有
        ↓
【鈴木】写真をCMSにアップロード＋本文を作成
        ↓
【鈴木】Slack「運用」チャンネルに共有
        ↓
【中野さん】テキストを確認・修正
        ↓
CMSステータスを「公開中」に変更して公開
        ↓
Slack「全体」チャンネルに公開を共有
```

---

## STEP 1｜事前準備（Studioを開く前に）

`授業記録テンプレート.md` を使って、以下の情報を手元に用意しておく。

| 項目 | 例 |
|------|-----|
| タイトル | 第1回前半講義　講師：横山 広美 先生 |
| 日付 | 2026年○月○日（曜）17時 |
| 場所 | 東京大学本郷キャンパス |
| 講師名・所属 | 東京大学国際高等研究所カブリ数物連携宇宙研究機構、学際情報学府教授 |
| 本文（2文） | 定型文①＋定型文② |
| 写真ファイル | YYYYMMDD_DSEP_第○回_01.jpg〜 |

---

## STEP 2｜CMSダッシュボードを開く

1. [studio.design](https://studio.design) にログイン
2. DSEPプロジェクトを選択
3. 左サイドバーまたは上部メニューから **「CMS」** を開く
4. 授業記録用のモデル（例：「授業記録」「講義レポート」など）を選択

---

## STEP 3｜新規アイテムを追加する

1. アイテム一覧画面右上の **「+ 新規追加」** をクリック
2. 新しい行（アイテム）が追加される
3. アイテムをクリックして詳細編集画面を開く

---

## STEP 4｜各プロパティに内容を入力する

### タイトル
```
第〇回前半講義　講師：○○ ○○ 先生
```
- 前半・後半は別アイテムとして作成する
- 式典の場合: `第○期 東京大学履修証明プログラム DSEPが開講しました！`
- 合宿の場合: `DSEPキャンプ2026を開催`

### 日付
```
2026年○月○日（曜）17時
```
- 複数日にまたがる場合: `2026年○月○日（曜）・○日（曜）`

### 本文（Body）

**通常講義の場合：**
```
2026年○月○日（曜）○○時より、東京大学本郷キャンパスにて、「災害対策エグゼクティブプログラム（DSEP）」第○回前半/後半講義として、[講師の所属・肩書]の○○ ○○先生にお話しいただきました。

○○先生はご自身の研究分野と絡めて、「○○○○」などについて話していただきました。
```

**開講式の場合：**
```
第○期 東京大学履修証明プログラム DSEPの開講式が執り行われました。

（式次第や内容を続けて記載）
```

---

## STEP 5｜写真をアップロードする

1. 画像プロパティの **「+ 画像を追加」** をクリック
2. 写真を複数枚アップロード（順番に注意）
3. ファイル名規則: `YYYYMMDD_DSEP_第○回_01.jpg`

**写真の選び方（優先順）：**
1. 登壇者の講義風景（メイン）
2. 受講者・会場全体の様子
3. ディスカッション・グループワーク
4. その他

---

## STEP 6｜プレビューで確認する

1. 詳細編集画面の **「プレビュー」** ボタンをクリック
2. 以下を確認する
   - [ ] タイトルの表示
   - [ ] 日付・場所の表示
   - [ ] 本文の改行・体裁
   - [ ] 写真の表示順・見切れなし
3. 問題があれば修正して再確認

---

## STEP 7｜公開する

1. ステータスを **「公開中」** に変更
2. サイト全体の再公開が必要な場合は右上 **「公開」** ボタンをクリック
3. 実際のURLで表示を最終確認

> **予約公開したい場合：** ステータスを「公開予約」にして公開日時を設定する

## Slack投稿テンプレート

### 運用チャンネルへの共有（鈴木 → 中野さんへ確認依頼）

```
【授業記録 CMS作成完了】第○回 前半/後半講義

記事を作成しました。テキストの確認・修正をお願いします。

■ 記事タイトル：第○回前半講義　講師：○○ ○○ 先生
■ 日付：2026年○月○日（曜）
■ プレビューURL：（Studio プレビューURLを貼る）

※確認OKでしたら、ステータスを「公開中」に変更してください。
```

### 全体チャンネルへの公開共有（公開後）

```
【DSEP 授業記録を公開しました】

第○回前半/後半講義「○○○○」の記録を公開しました。

■ 講師：○○ ○○ 先生
■ 公開URL：（公開URLを貼る）
```

## よくあるミス・注意点

| 状況 | 対処 |
|------|------|
| 公開したのに表示されない | ステータスが「下書き」のままになっていないか確認 |
| 写真が表示されない | サイト全体を再公開する |
| 前半・後半を1記事にまとめてしまった | 別アイテムに分割して作成しなおす |
| 講師の所属が略称になっている | 正式名称で記載する（例：「東京大学国際高等研究所カブリ数物連携宇宙研究機構、学際情報学府教授」） |
