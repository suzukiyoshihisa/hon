# 岡崎食品 EC コンサルティング計画

## 概要

- **目的**: EC作業（Amazon・楽天への出品代行）を起点に、岡崎食品のオンライン販売全体を支援・強化するコンサルティングを実施する
- **契約**: 月額19万円（運用代行込み）
- **HONの役割**: EC戦略立案・実行支援・数値管理・改善提案

---

## コンサルティングの考え方

EC作業は「手段」であり、目的は **岡崎食品の商品が正規ルートで消費者に届き、売上と信頼が積み上がること**。
作業の進捗・数値をもとに毎月振り返りを行い、次の施策をクライアントと一緒に考える伴走型コンサルとする。

---

## フェーズ構成

### Phase 1｜基盤整備（1〜2ヶ月目）

**EC作業（実務）**
- Amazon セラーセントラル アカウント開設代行
- 商品ページ作成（タイトル・説明文・箇条書き・画像ディレクション）
- ブランドレジストリ申請
- 楽天市場 出店申請・ページ作成

**コンサル（方針策定）**
- 現状分析：無断出品の棚卸し・既存ASINの確認
- 商品ラインナップの整理（何を・どの順で・どの価格で出すか）
- 競合調査（類似商品の価格帯・レビュー数・差別化ポイント）
- ブランドコンセプトの言語化（タイトル・説明文の方向性を統一）

**月次レポート項目**
- 出品状況（進捗）
- 競合ベンチマーク

---

### Phase 2｜正規化・信頼構築（2〜4ヶ月目）

**EC作業（実務）**
- 無断出品ページへのブランドレジストリ経由の削除申請
- 正規ページへの購買集約
- Amazon Ads（スポンサープロダクト広告）初期設定・運用開始

**コンサル（改善提案）**
- 商品ページの改善提案（CTR・CVRのデータを見ながら）
- レビュー獲得施策の設計（同梱物・フォローアップ）
- 価格戦略の見直し（競合対比・利益率チェック）

**月次レポート項目**
- 売上・注文数・セッション数・CVR
- 広告費用対効果（ROAS・ACoS）
- レビュー件数推移
- 無断出品の対応状況

---

### Phase 3｜成長・拡大（5ヶ月目〜）

**EC作業（実務）**
- 広告予算最適化・入札調整
- 新商品の追加出品
- セール・クーポン施策の実施

**コンサル（戦略拡張）**
- 楽天・Amazon の売上比較分析と注力チャネルの判断
- 新商品開発へのEC視点からのフィードバック（売れ筋・レビューの声を反映）
- 自社ECサイト（BASE / Shopify など）への展開検討
- SNS・コンテンツマーケティングとの連携提案

**月次レポート項目**
- チャネル別売上比較
- 新商品パフォーマンス
- 中長期ロードマップ更新

---

## 月次コンサルMTG（想定）

| 項目 | 内容 |
|------|------|
| 頻度 | 月1回（60〜90分） |
| 形式 | オンライン（Zoom等）または現地 |
| アジェンダ | ①先月レポート共有 → ②課題・質問対応 → ③翌月施策の合意 |
| 資料 | HONが月次レポートを事前共有 |

## KPI（目標指標）

| 指標 | Phase 1 目標 | Phase 2 目標 | Phase 3 目標 |
|------|-------------|-------------|-------------|
| 正規ページ出品数 | 主力3商品 | 全商品 | 新商品追加 |
| 月間売上（Amazon） | 立ち上げ | 5万円〜 | 20万円〜 |
| レビュー件数 | 0→獲得開始 | 各商品10件〜 | 各商品30件〜 |
| 無断出品の排除 | 棚卸し完了 | 主要品削除申請済み | 完全排除 |

---

## 確認事項（岡崎側への宿題）

- [ ] 商標登録の有無（ブランドレジストリ申請に必要）
- [ ] JANコードの有無
- [ ] 商品写真の素材（高解像度データ）
- [ ] 楽天市場への出店意向・予算感
- [ ] 現在の主な販売先・売上比率
