## 記事作成の手順

### Step 1: テーマを受け取る
鈴木からLINEでテーマと一言メモが届く。例：
```
テーマ：「ブランドロゴを自分で作るべきか外注すべきか」
方向性：初期コスト感と将来的なブランド価値の観点で整理してほしい
```

### Step 2: Claudeに記事を書かせる

以下のテンプレートをClaude.aiに貼り付けてテーマを差し替えて使う。

---

**Claudeへの指示テンプレート（コピー用）**

```
あなたはブランド支援会社「WANSIE DESIGN」のライターです。
小規模ブランドや個人起業家向けに、デザイン・Web・SNSのノウハウを発信しています。

以下のテーマでnote記事を書いてください。

【テーマ】
（ここにテーマを入れる）

【条件】
- 文字数：1,000字前後
- 読者：ブランドを立ち上げたばかり、またはこれから立ち上げる個人・小規模事業者
- トーン：親しみやすく、実践的。難しい専門用語は避ける
- 構成：見出しを2〜3つ使って読みやすくする
- 最後に一言まとめ（2〜3文）で締める
- 自社サービスの宣伝は入れない

【参考にしてほしい方向性】
（鈴木からのメモをここに貼る）
```

---

### Step 3: 出力を確認・調整する

Claudeの出力をそのまま使わず、以下をチェックする：

- [ ] 1,000字前後に収まっているか（多すぎたら削る）
- [ ] 難しい言葉や業界用語が混ざっていないか
- [ ] 「〜かもしれません」「〜と思います」が多すぎないか（断言に直す）
- [ ] 読んでいて自然な流れか（声に出して読んでみる）

気になる部分はClaudeに「この部分をもっと〇〇な表現に直して」と追加で指示して修正する。

### Step 4: noteに下書き保存

1. note.com にログイン（アカウント情報は鈴木から共有）
2. 「新しい記事を書く」→ 本文を貼り付け
3. タイトルを設定（Claudeに「タイトル案を3つ出して」と聞くと楽）
4. **「下書き保存」で保存**（公開しない）
5. 鈴木にLINEで「下書き保存しました」と連絡

## 品質基準

**OK**
- 読んで「なるほど」と思える具体的な内容がある
- 1,000字前後に収まっている
- 見出しで内容が予測できる

**NG（鈴木が修正して返す）**
- Claudeの出力をそのままコピペしただけ（修正の跡がない）
- 抽象的な励ましで終わっている（「頑張りましょう！」など）
- 誤字・脱字がある
