# DSEP 第3期 参加者限定ページ 構成案

作成日: 2026-04-30
作成: 株式会社HON 鈴木
対象: 沼田教授・DSEP運営チーム

---

## 概要

第3期参加者（エグゼクティブ）が「このページだけ見れば授業の全情報がわかる」ワンストップページ。
Studioのパスワード保護機能で限定公開し、参加者にのみURLとパスワードを共有する。

---

## アクセス制限の設計

| 項目 | 設定 |
|------|------|
| 公開範囲 | Studioのページパスワード保護（ページ単位） |
| URL形式 | `https://dsep-tokyo.studio.site/members` など |
| パスワード | 期ごとに設定（例：第3期固有のパスワード） |
| 共有方法 | 開講式時にメール or Slackで案内 |

> **注意:** Studioのパスワード保護はページ全体に1つのパスワードをかける方式。
> 個人ごとのログインはできないため、パスワードの取り扱いを参加者に周知する。

## ページ構成

### ファーストビュー（常に最上部に表示）

```
┌─────────────────────────────────────────────┐
│  DSEP 第3期 参加者ページ                        │
│  東京大学履修証明プログラム                       │
├─────────────────────────────────────────────┤
│  🔴 LIVE配信中  ／  📅 次回授業：〇月〇日（曜）   │  ← 状態に応じて切り替え
└─────────────────────────────────────────────┘
```

- LIVE配信中はバナーを赤（or ゴールド）で目立たせ、URLをすぐ押せる状態に
- 配信がない日は「次回授業の日時・講師」を表示

### セクション構成（上から優先度順）

---

#### ① 次回授業（最重要・最上部）

参加者が最も頻繁に確認する情報。常にここを更新する。

| 表示項目 | 内容 |
|---------|------|
| 日時 | 〇年〇月〇日（曜）17:00〜19:00 |
| 場所 | 東京大学本郷キャンパス / オンライン |
| 講師 | 〇〇 〇〇 先生（〇〇大学 〇〇研究所 教授） |
| テーマ | 〇〇〇〇 |
| 事前資料 | PDFリンク or 「なし」 |
| YouTube Live URL | 配信URLをボタンで表示（当日更新） |
| 備考 | 持ち物・注意事項など |

**更新タイミング:** 授業の1週間前までに更新。YouTube Live URLは前日〜当日朝に追加。

---

#### ② 授業スケジュール一覧

第3期の全授業をテーブルで表示。進捗が一目でわかるステータス付き。

| # | 日付 | 時間 | 講師 | テーマ | 場所 | ステータス |
|---|------|------|------|--------|------|-----------|
| 第1回 | 〇月〇日（曜） | 17:00〜 | 〇〇 先生 | 〇〇〇〇 | 本郷 | ✅ 終了 |
| 第2回 | 〇月〇日（曜） | 17:00〜 | 〇〇 先生 | 〇〇〇〇 | オンライン | 📅 次回 |
| 第3回 | 〇月〇日（曜） | 17:00〜 | 〇〇 先生 | （調整中） | 未定 | — |
| ... | | | | | | |

**Studio実装:** CMSコレクション「授業スケジュール」で管理。
各行をクリックすると事前資料・配信URLへ遷移。

## 更新フロー（限定ページ専用）

### 定期更新（授業1週間前）

```
【鈴木 or 中野さん】
授業スケジュールCMSを更新
  → 講師・テーマ・事前資料を入力
  → Slack「運用」で更新完了を共有
```

### 当日更新（YouTube Live URL）

```
【当日朝〜1時間前】
YouTubeでLiveイベントを作成 → URLをコピー
  → 授業スケジュールCMSの「YouTube Live URL」フィールドに入力
  → 即時公開（参加者が当日確認できる状態に）
```

### 名簿更新（開講時 + 随時）

```
【開講式後】
参加者情報を収集（許諾確認済み）
  → 参加者CMSに一括入力
  → 限定ページで公開
```

## 補足：今後の拡張アイデア

- **アンケートへの導線:** 各授業後にGoogleフォームのリンクをここから共有
- **修了証の案内:** 修了要件（出席回数など）をこのページに掲載
- **期をまたいだ交流:** 2期参加者紹介へのリンクを追加（コミュニティ感の醸成）
