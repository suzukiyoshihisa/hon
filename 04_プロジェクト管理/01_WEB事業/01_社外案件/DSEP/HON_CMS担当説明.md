# 授業記録の発信
授業記録のCMS作成・WEB掲載をワンストップで担当します。

---
## DSEP運営の方がやること
| 作業 | 内容 |
|------|------|
| 掲示板的に見せたい情報（鍵付きOK） | LINEなどで整形していない情報送る |
| 💬 Slackへの共有 | 掲載内容をSlackにわかりやすく投稿 |
| ❓ 質問対応 | 内容の確認・修正もSlack上で対応 |


## 鈴木がやること
| 作業 | 内容 |
|------|------|
| 📸 写真の受け取り | カメラマンから直接受け取り・整理 |
| ✍️ 記事の作成 | テキストを整形してCMSに入力 |
| 🎨 デザイン調整 | 見栄えよく掲載（写真順・文字量など） |

---

## Before / After

### Before（鈴木なし）

```
運営スタッフ
  → 写真をカメラマンから受け取る
  → CMS操作を覚える・入力する
  → 文章を整える
  → 写真を選んでアップロード
  → 公開作業
  → Slackに共有
```
→ **運営側に負担が集中**

### After（鈴木あり）

```
運営スタッフ
  → テキストをLINE等で鈴木に送る　← これだけ！
        ↓
      鈴木が全部やる
        ↓
      運営側は確認して、slackに投稿するだけ
```
→ **運営側はゼロ負担に近い**

---

## 鈴木に任せる3つの理由

### 1｜WEBの専門家
Studio CMSの操作・デザイン調整・公開作業を熟知。運営の方が操作を覚える必要なし。

### 2｜カメラマン・中野さんと直接連携できる
写真の受け渡しや情報共有ハブとなって行います。運営スタッフが間に入る手間がなくなります。

### 3｜デザイン・情報整理の知見がある
テキストをそのまま貼るのではなく、読みやすく・見栄えよく整えて掲載します。
講師の所属表記や写真の選び方など、細かい品質管理も担当します。

---

## 連絡方法

| 手段 | 用途 |
|------|------|
| LINE / メッセージ | 授業後のテキスト共有（気軽にどうぞ） |
| Slack | 作成した記事の共有・確認・質問対応 |

**送ってほしい情報（最低限）：**
- 授業日・講師名
- 講師の所属・肩書
- 授業で話されたこと（箇条書きでもOK）

写真はカメラマンから直接受け取ります。
