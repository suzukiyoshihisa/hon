# カオリウム キャッシュポイント

## 全体の収益フロー
法人でのワークショップにつなげるために以下の施策を計画している

```
施策①　YouTube Live（毎週火曜）
　└─ 認知獲得・視聴者との関係構築（直接収益なし）
　　　└─ 施策②　エグゼクティブ向けトークセッション＋交流会（個人参加・¥30,000）
　　　　　└─ 施策③ 法人契約へ転換（香り会議・法人向けWS）
```

3つの施策は独立しているのではなく、**YouTube Liveで興味を持った人をワークショップに誘導し、そこから法人契約につなげる**一連の流れです。

---

## 施策①：YouTube Live（認知・集客チャネル）

**目的**: カオリウム体験に興味を持つ人を継続的に集め、ワークショップへの参加につなげる

> **この施策の位置づけ**: 直接の収益は発生しない。ゲストの発信力と視聴者との信頼関係を積み上げることで、ワークショップ参加者を安定的に集めることが目的。

| 項目 | 内容 |
|------|------|
| 頻度 | 毎週火曜日 |
| チャンネル | Magnet lab（kINGさんが運営するチャンネル） |
| 位置づけ | カオリウム公式ではなく、Magnet labがカオリウムをコンテンツの素材として活用するチャンネル。 |

### 配信の構成

| # | コンテンツ | 役割 |
|---|-----------|------|
| 1 | 挨拶・自己紹介 | 視聴者との関係づくり |
| 2 | カオリウムの紹介 | 毎回説明することで新規視聴者にも届ける |
| 3 | テーマ体験「カオリウムで○○を言語化する」 | メインコンテンツ・ゲストと一緒に毎回テーマを変える |
| 4 | テーマに関する雑談・深掘り | 共感・信頼の醸成 |
| 5 | 食事会（ネットワーキング）のご案内 | ワークショップへの誘導 |

### 出演者の役割分担

| 担当 | 役割 |
|------|------|
| kINGさん | Magnet labチャンネルのホスト・主催。配信全体の進行を担う |
| ゲストさん | カオリウムを使ってテーマ（○○）を言語化する体験をリアルに見せる役 |
| 鈴木 | カオリウムの紹介・解説、場の補助 |
| たずさん | 配信設定・技術面の運営 |

### ゲスト候補

| 優先度 | 候補者 |
|--------|--------|
| 高（初回〜近日） | 松本めぐみさん |

## 施策②：エグゼクティブ向けトークセッション＋交流会（直接収益＋法人開拓）

**目的**: 個人参加費で収益を得ながら、法人契約への入口をつくる

> **この施策の位置づけ**: 参加者のネットワーキングと**法人契約獲得のための入口**として位置づける。収益の本命は、ここから生まれる法人向けワークショップの継続契約。

### 収益構造

| 区分 | 項目 | 金額 |
|------|------|------|
| **売上** | 参加費（約10名） | 約3万円 × 10名 ＝ **約30万円** |
| **コスト** | セントマティック（機材・運営） | 一 |
| **粗利** | | 30万円(仮) |

### 出演者の役割分担

| 担当 | 役割 |
|------|------|
| kINGさん | ファシリテーター（主催）|
| 谷本さん | ファシリテーター（主催） |
| ゲストさん | カオリウムを使ってテーマ（○○）を言語化する体験をリアルに見せる役 |
| 鈴木（HON） | サポート役として参加（「たずさん」ポジション）|

### 法人契約への営業導線

```
ワークショップ参加（エグゼクティブ個人）
　│
　├─ 谷本さんのファン　→ 谷本さんが直接フォロー・法人提案
　├─ その他の参加者　　→ サポート役がフォロー・法人提案
　│
　▼
「ご自身の会社でも体験会を開きませんか？」と提案
　│
　├─ 会社説明用の資料を作成・送付
　├─ 香り会議（法人向けワークショップ）を実施
　│
　▼
顧客管理・継続フォローアップ
```

## 施策③：法人契約への転換（本命の収益）

**目的**: トークセッションで接点を持った参加者の所属企業に、カオリウムを使った法人向けワークショップを提案・契約する

> **この施策の位置づけ**: ここが収益の本命。施策①②はこの契約獲得のための導線として機能する。

### 収益構造（仮）

| 区分 | 項目 | 金額 |
|------|------|------|
| **売上** | 参加費 | ¥20,000 / 人（松本興産様の実績より） |

### 提供内容

| メニュー | 内容 |
|----------|------|
| 香り会議 | 企業の会議やチームビルディングにカオリウムを組み込んだワークショップ |
| 法人向けWS | 企業理念・ブランド構築・組織の相互理解を目的にした本格ワークショップ |

### 営業の流れ

```
施策②の参加者（エグゼクティブ個人）
　│
　▼
「ご自身の会社でも体験会を開きませんか？」と提案
　│
　├─ 会社説明用の資料を作成・送付
　│
　▼
香り会議（法人向けワークショップ）を実施
　│
　▼
継続契約・顧客管理（フォローアップ）
```
