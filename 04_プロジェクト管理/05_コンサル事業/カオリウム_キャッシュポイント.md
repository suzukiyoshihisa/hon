## 1. YouTube Live を使った営業活動
- **開催頻度**: 毎週火曜日
- **チャンネル**: HON独自配信（※カオリウム公式YouTubeではない。カオリウムはあくまで「借り物」として活用）

### 配信の流れ
| 順番 | 内容 |
|------|------|
| 1 | 挨拶 |
| 2 | 自己紹介 |
| 3 | カオリウムの紹介 |
| 4 | テーマ「カオリウムで○○を言語化する」 |
| 5 | 雑談 |
| 6 | 終わり |
| 7 | **食事会（ネットワーキング）** |

## 2. CtoC エグゼクティブ向けワークショップ
谷本さんを主軸にした、エグゼクティブ向けの対面ワークショップ。

### 収益構造【キャッシュポイント】
| 項目 | 金額・規模 |
|------|------------|
| セントマティック（機材・運営） | 約30万円 |
| 谷本さん（ファシリテーション） | — |
| 参加費 | 1人 約3万円 × 約10人 |
| ネットワーキング食事会 | 参加費に含む or 別途 |

### 営業導線（ワークショップ → 法人契約）
```
ワークショップ参加（エグゼクティブ個人）
　│
　├─ 谷本さんファン → 谷本さんが法人営業
　├─ その他        → ソフィアさん等が法人営業
　│
　▼
「自分の会社でもやりませんか？」と提案
　│
　├─ 会社説明用の資料作成
　├─ 香り会議（法人向けワークショップ）
　│
　▼
Salesforce で顧客管理・フォローアップ
```

### 体制メモ
- 谷本さん主催の時は、鈴木が「たずさん」のポジション（サポート役）に入ることも可能
