## デモの狙い

- 岡田豊社労士事務所（現サイト: adjust-srf.jp）のリニューアル版モック
- 見せ場は1つ：**「初回60分相談」がオンラインで予約できる**こと。デザインの綺麗さより「予約→自動で日程確定」の体験を優先
- 完璧な完成品ではなく「自分の事務所がこうなるのか」と想像させる提案ツール

## サイト構成（4ページ＋予約フロー）

```
TOP
├── 初回相談（フロント商品ページ）★デモの主役
├── サービス・料金
├── 事務所・代表プロフィール
└── お問い合わせ
```

## 各ページの構成とテキスト案

### 1. TOP

- ヒーローコピー（案）:
  > **「人の悩み、60分で道筋を。」**
  > 飲食店経営30年 × 社労士。現場を知る専門家が、労務の不安を整理します。
- ファーストビュー直下にCTAボタン: **［初回相談を予約する（60分）］**
- 続けて3ブロック: ①こんなお悩みありませんか（採用・勤怠・助成金・就業規則）②初回相談で得られるもの ③代表の顔写真＋一言

### 2. 初回相談ページ（フロント商品）★

- 商品名（案）: **初回相談60分パック**
- 提供内容（既存メニューをパッケージ化して見せる）:
  1. 60分の対面/オンライン相談
  2. **労務診断**（現状の労務リスクを簡易チェック）
  3. **助成金提案書**（使える可能性のある助成金の一覧を後日お渡し）
- 価格: 【要確認：岡田さんの現行価格。デモでは仮で¥11,000（税込）と表示】
- 流れ: 予約カレンダーで日時選択 → 事前アンケート（従業員数・業種・相談したいこと）→ 当日相談 → 1週間以内に診断結果＋提案書
- CTAボタン: ［空き日程を見る］

### 3. サービス・料金

- 顧問契約／スポット（就業規則・助成金申請・給与計算）を表で。価格は【要確認・仮表示でOK】
- ページ末尾に「まずは初回相談から」導線を必ず置く

### 4. 事務所・代表プロフィール

- ストーリー構成（案）: 飲食店経営30年 → だから中小企業・店舗の労務の現場がわかる → 社労士として独立
- 【要確認：経歴の事実関係・写真素材】

## Shopify実装メモ

| 項目 | 方針 |
|------|------|
| プラン | デモ段階は開発ストア（無料）でOK |
| 予約機能 | 予約アプリを使用（Appointment Booking系の無料枠。候補2〜3比較して選定——本番でも使うのでここは丁寧に） |
| 商品登録 | 「初回相談60分パック」を商品として登録（在庫∞・所要60分） |
| 決済 | デモでは決済まで通さなくてよい（予約確定画面まで見せられれば十分） |
| 参考 | 現サイト adjust-srf.jp の情報を流用。ロゴ・写真は仮でOK |
